Communication Method (Unidirectional Printing)
The printer communication method is unidirectional printing.
Unidirectional Printing
Data can be sent only from the computer to the printer.
The following functions that require bidirectional communication are not available:
Acquiring information regarding options installed in the printer etc.
NOTE
To use the options set in the printer you are using, you must set the options at the computer. For details about setting the options, see Printer Driver Installation Guide.
Selecting custom paper types registered in the printer
