[Tab Position Settings] dialog box
To display the [Tab Position Settings] dialog box, click [Tab Position] in the [Paper Source] preferences pane.
[Unit]:
Selects the unit of measurement used to specify the shifting width.
[Shifting Width]:
Specifies the width for shifting the image data to be printed on the tab part of tab paper.
