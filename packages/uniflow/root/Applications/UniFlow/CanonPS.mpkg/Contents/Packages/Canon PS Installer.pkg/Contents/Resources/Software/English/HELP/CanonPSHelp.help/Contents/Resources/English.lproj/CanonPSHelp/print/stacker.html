[Stacker Setting] dialog box
To display the [Stacker Setting] dialog box, click [Stacker Setting] in the [Finishing] preferences pane.
[Specify Number of Copies to Stack]:
If you want to specify a number of copies to stack, select this option.
[Number of Copies to Stack]:
Enables you to specify the number of pages when outputting a number of copies.
