[Color Matching] preferences pane (Mac OS X version 10.5 and later)
This pane lets you apply color profiles.
[ColorSync]:
Color is controlled from the application.
[In Printer]:
Color is controlled from the printer.
[Profile]:
You can specify a color profile you want to print with.
