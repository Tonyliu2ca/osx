[Toner Density Adjustment] dialog box
To display the [Toner Density Adjustment] dialog box, select [On] for [Toner Density Adjustment] then click [Settings] in the [Quality Settings] dialog box. You can adjust the toner density for each color. Click or drag the slider bar to adjust the density.
