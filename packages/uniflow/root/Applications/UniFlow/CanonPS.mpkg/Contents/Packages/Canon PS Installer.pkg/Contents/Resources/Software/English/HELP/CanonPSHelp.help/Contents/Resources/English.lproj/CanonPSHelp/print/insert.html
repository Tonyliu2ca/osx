[Inserting Sheet Settings] dialog box
To display the [Inserting Sheet Settings] dialog box, click [Inserting Sheet Settings] in the [Paper Source] preferences pane.
[Settings List]:
Displays the list of sheets for insertion you have added.
[Delete]:
Clicking this button deletes the selected entry in the [Setting List].
[Sheet for Insertion]:
Specifies the type of sheet to be inserted into the document.
[Paper Source]:
Specifies the paper source to use for inserted sheets, chapter pages, and tab paper.
[Print on]:
Specifies whether to print on inserted sheets, chapter pages, or tab paper for insertion.
NOTE
If you are inserting a chapter page, the front side is always printed.
[Insert]:
Specifies the page numbers of documents at which to add inserted sheets, chapter pages, or tab paper. Most printer models allow up to 20 sheets to be inserted into a document, but up to 100 sheets are also possible, depending on the printer model.
NOTE
Most printer models allow up to 20 sheets to be inserted into a document, but up to 100 sheets are also possible, depending on the printer model.
[Tab Paper]:
Enables you to specify the type of tab paper and the width for shifting the image data to be printed on the tab part of the paper.
[Paper Type]:
Displays the paper type of tab paper.
[Settings]:
Specifies the paper type of tab paper.
[Tab Paper Details]:
Opens a dialog box to specify the width for shifting the image data to be printed on the tab part of tab paper. You cannot specify the unit of measurement.
[Add]:
Fixes the insertion positions you specified. When the sequence for sheet insertion differs from that of the insertion positions, these are automatically modified in ascending sequence.
