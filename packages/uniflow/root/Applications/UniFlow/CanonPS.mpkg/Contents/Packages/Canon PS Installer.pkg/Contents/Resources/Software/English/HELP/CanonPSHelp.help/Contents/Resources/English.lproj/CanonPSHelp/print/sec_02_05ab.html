Performing Partial Clear Coating Printing (Combining)
1.
Display the document you want to print in the application.
2.
Select [Print] from the [File] menu.
The [Print] dialog box is displayed.
3.
In the [Print] dialog box, select the [Special Features] preferences pane.
The [Special Features] preferences pane is displayed.
4.
Click [Page Options].
The [Page Options] dialog box is displayed.
5.
Select [Clear Coating on Form Composition] from [Processing Method].
6.
Select the form file to be used from [File List].
7.
Select the form composition method from [Application].
If you want to apply clear coating with the same form file for all pages, select [Same for All Pages], and if you want to multiple form files to multiple pages with different content, select [By Page].
8.
Click [OK] in the [Page Options] dialog box.
9.
Click [Print].
The form file is combined with the document, and a printed document with clear coating applied is output from the printer.
