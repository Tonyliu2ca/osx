Halftone Settings
This section describes how to specify the halftone processing method for printing a document.
1.
From the [File] menu of the application software, select [Print].
The [Print] dialog box is displayed.
2.
Select [Quality] to display the [Quality] preferences pane.
3.
Click [Quality Settings].
The [Quality Settings] dialog box is displayed.
4.
Select an appropriate halftone setting from [Halftones] → click [OK].
NOTE
For details about halftone settings, see "[Quality Settings] dialog box."
5.
Click [Print] in the [Print] dialog box.
