Specifying Output Options
For some printers, you can specify the paper output destination for printing.
NOTE
The printer driver images shown here may differ from the actual printer driver.
1.
From the [File] menu of the application software, select [Print].
The [Print] dialog box is displayed.
2.
Select the [Finishing] preferences pane.
The [Finishing] preferences pane is displayed.
3.
Select a paper output destination.
4.
Click [Print].
