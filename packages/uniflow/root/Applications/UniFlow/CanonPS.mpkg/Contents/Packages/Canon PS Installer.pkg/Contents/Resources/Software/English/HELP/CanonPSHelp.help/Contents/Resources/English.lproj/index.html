About the PS Printer Driver
What is the Printer Driver?
The printer driver is used when printing documents.
The printer driver converts the print data generated by applications into a form readable by the printer, and then sends it to the printer.
The printer driver should be compatible with the printer used because the data available to print differs according to the printer model.
Features of the PS Printer Driver
The following are the features of the PS Printer Driver.
Prints on various paper sizes.
Prints enlarged or reduced documents.
Prints multiple pages on one page, or prints on both sides of the paper.
Prints overlays.
Saves printer driver settings as a Profile for printing documents under specific settings.
Specifies various output destinations for the print job.
Uses the Job Accounting feature.
Adds front and back covers, sheet insertions, and chapter pages to the document.
Inserts or prints on tab paper (available depending on the printer controller version).
Differences in Functions of Various Printer Models
All the functions of the PS printer driver are described in this online help. The functions that are available for the printer you are using depend on the printer model, function version, and installed options (finisher, hard disk, etc.). To find out which functions are available for the printer you are using, refer to the instruction manual provided with the printer.
