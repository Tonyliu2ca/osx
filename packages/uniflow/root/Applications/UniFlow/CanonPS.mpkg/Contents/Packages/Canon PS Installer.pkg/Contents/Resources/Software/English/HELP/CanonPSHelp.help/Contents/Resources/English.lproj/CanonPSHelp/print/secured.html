[Secured Print] dialog box
Enables you to set a password for print jobs and store them in the printer. To output a print job, enter the password at the printer. This function is useful when you want to print confidential documents.
[Document Name]:
Specifies the name of the current document.
[User Name]:
Specifies the user name for secured printing.
NOTE
The maximum number of characters allowed for [Document Name] and [User Name] differs depending on the printer model you are using.
[Password]:
Specifies the password for secured printing. You can enter a numerical password from 0 to 9999999. Outputting data requires the password you have set here.
