[Front/Back Cover Settings] dialog box
To display the [Front/Back Cover Settings] dialog box, click [Front/Back Cover Settings] in the [Paper Source] preferences pane. This dialog box enables you to set a paper source other than that set in the [Paper Source] preferences pane for the front and back covers. You can also select a print side other than that set in the [Finishing] preferences pane.
[Cover Settings]:
Enables front and back covers to be added to the document.
[Paper Source]:
Specifies the paper source to use for printing the front and back covers.
[Print on]:
Specifies whether to print on the front and back covers of the document. If you select [None], a blank page will be inserted.
