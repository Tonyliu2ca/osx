[Scheduler] preferences pane
This pane lets you specify a time to start printing and the printing priority.
NOTE
In Mac OS X version 10.5 and later, depending on the application, the settings in each preferences pane are displayed at the bottom.
[Print Document]:
Specifies when to print the document.
[Now]:
Starts printing immediately.
[At]:
Specifies the time to start printing.
[On Hold]:
Puts the printing process on hold.
[Priority]:
Select the printing priority from [Urgent], [High], [Medium], and [Low].
